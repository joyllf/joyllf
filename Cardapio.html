Passo a passo de praparar um café perfeito.

  1. Ferva a água(90°C).
  2. Adicione o pó café no coador.
  3. Despeja a água quente sobre o pó dentro do coador.
  4. Armazene em uma garrafa térmica ou coloque em um copo.

explicando o que são HTML, CSS e JavaScript.

O que é HTML?
É a linguagem padrão para criar a estrutura de páginas da web, definindo o conteúdo como títulos, parágrafos, imagens e links.
O que é CSS
É uma linguagem de estilo usada para descrever a aparência de documentos escritos em linguagens de marcação como HTML.
O que é JavaScript
é uma linguagem de programação usada principalmente para adicionar interatividade e dinamismo a páginas web, manipulando o conteúdo de forma dinâmica.
